# Daniel Web Development Portfolio

## Install Dependencies

```bash
# NPM
npm install

# YARN
yarn i

# PNPM
pnpm i

# BUN
bun i
```

## Run in development mode

Open the browser at http://localhost:3000/

```bash
# NPM
npm run dev

# YARN
yarn dev

# PNPM
pnpm dev

# BUN
bun dev
```

## Build the project

```bash
# NPM
npm run build

# YARN
yarn build

# PNPM
pnpm build

# BUN
bun build
```
